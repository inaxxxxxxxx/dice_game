# チンチロリン

ブラウザで遊べるチンチロ（3つの骰子を使った賭けゲーム）の Web アプリです。
骰子は Three.js による 3D で、碗（茶碗）の中で物理っぽく転がって止まります。

## 遊び方

1. 参加人数（2〜6人）と、負けが確定する人数を決める
2. 各プレイヤーが順番に端末を受け取り、名前を入力（空欄なら「プレイヤーN」が入ります）
3. 各プレイヤーが骰子を振る。目が出るまで最大3回まで振り直せる
4. 全員が振り終えたら役を判定し、弱い方から負け確定人数分だけ負けが決まる
5. 同着で負け枠が決まらない場合は、サドンデス画面で対象者を表示し、その対象者だけで再戦
6. 勝敗が確定したら結果画面を表示

ゲーム中は左上の「✕ TOP」ボタンでいつでもタイトルに戻れます（確認ダイアログあり）。

## 役の強さ（強い順）

1. ピンゾロ（1・1・1。最強特殊役）
2. アラシ（同じ目が3つ。ゾロ目同士は数字が大きい方が勝ち）
3. シゴロ（4・5・6）
4. 通常の目（2つ同じ目＋1つ違う目。違う目の数字が大きい方が勝ち）
5. 目なし（3つとも異なる目で役なし）
6. ヒフミ（1・2・3。反則目で最弱・即負け扱い）

## モード

タイトル画面下の「確率テーブル」リンクから、確率の詳細を確認できます。  
このページで **デフォルト** と **甘チンモード** を切り替えられます。甘チンモードでは演出の出現率と良い役の確率が上がり、サイコロを振るボタンが「サイコロを振る（甘チン）」になります。

## 演出と出目の確率ロジック

サイコロを振るたびに一定確率で演出（炎・雨・泡・魚群）が発動し、出目が確率的に補正されます。  
デフォルトは演出 **10%** ／通常サイコロ **90%**、甘チンモードは演出 **40%** ／通常 **60%**。  
各演出には Web Audio API で合成した効果音（炎＝ボーボー、雨＝ザーザー、泡＝ゴボゴボ、魚群＝水中音）が付きます。

### 特殊役

- **ピンゾロ**：1が3つ揃った場合の最強役。アラシより上の独立したランク。
- **ヒフミ**：1・2・3の反則目。すべての役より弱く即負け扱い。

## ファイル構成

```
index.html   画面構成（タイトル〜結果まで全画面）／OGP・アイコン設定
style.css    見た目（モダン・ミニマル、白基調）
game.js      ゲームフロー制御・役判定・勝敗ロジック・モード/演出の確率制御
dice3d.js    Three.js による3D骰子の物理アニメーション
effects.js   演出（炎・雨・泡・魚群）の Canvas 描画
sounds.js    Web Audio API による効果音合成（外部音源なし）
```

ビルド不要。`index.html` を直接ブラウザで開くか、簡易サーバーで配信するだけで動きます。

## ローカルでの動かし方

```bash
npm run dev
```

`http://localhost:8910` で開きます（`python3` が必要です）。

Node 製のサーバーを使いたい場合は `npx serve .` でも代用できます。

## 既知の調整ポイント

- 骰子の重さ・弾み方は `dice3d.js` の `GRAVITY`（現在 `-34`）・上向き初速・反発係数（`* -0.32` の箇所）で調整できます。
- 役の強さの序列は `game.js` 内の `HAND_RANK` で定義しています。
- 演出の出現率・出目補正テーブルは `game.js` 内の `MODE_CONFIG`（`default` / `amachin`）で定義しています。
